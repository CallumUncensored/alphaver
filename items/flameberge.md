# Flameberge
| Item ID | In-game Name |
| ------- | ------------ |
| 362     | Flameberge   |

## Obtaining The Item
The Flameberge can be obtained by killing a giant.  
Read more about it [on the page about giants](../mobs/LilypadGiant.md).

## Special Abilities
By right-clicking, a ring of fire will spawn around you.
